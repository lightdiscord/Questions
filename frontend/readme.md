## Quick Start

Project setup
```console
$ npm install
```

Compiles and hot-reloads for development
```console
$ npm run serve
```

Compiles and minifies for production
```console
$ npm run build
```

Lints and fixes files
```console
$ npm run lint
```

Run your unit tests
```console
$ npm run test:unit
```
